------------Pseudo Classes------------ ->A pseudo class in CSS is used to define the special state of an element. ->It can be combined with a CSS selector to add an effect to existing elemnets based on thier states. ->For Example,changing the style of an element when the user hovers over it,or when a link is visited. ->All of these can be done using pseudo classes in CSS. Note:That pseudo-class names are not case-sensitive. Syntax: selector:pseudo-class { property:value; } ->There are many pseudo-classes in CSS but some of the most commonly used pseudo-classes are: *:hover -It is used to select and style an element when the user hovers over it with a pointing device. *:active -It is used to select and style an element when it is being activated by the user. *:focus -It is used to select and style an element when it has focus. *:visited -It is used to select and style a link that has been visited by the user. ------------Pseudo Elements------------ ->A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected elements. ->For Example,styling the first letter or line of an element. ->Insert content before or after the content of an element,all of these can be done by using the pseudo-elements in CSS. Note:that in contrast to pseudo-elements,pseudo-classes can be used to style an element based on its state.
