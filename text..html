Regular function Expression: JavaScript Let calculatorSum = function(x,y) { return x + y; }; console.log(calculatorSum(12,4)); output:16 Arrow Function JavaScript Let calculateSum = (x,y)=> { return x + y; }; console.log(calculateSum(12,4)); output: 16 Note:In the arrow function, we replace the function keyword with the arrow syntax =>. *Simplifying Arrow Functions ->Arrow functions can be further simplified when they have a single expression to return. ->In such cases, the return keyword and the curly braces{} can be omitted.
